Enviro

Command-line Options

-no_plants This will suppress loading of the plant library; plants will not be available, but Enviro launches faster.
-no_startup_dialog This will suppress the startup dialog, Enviro will simply use the contents of the .ini files as if the dialog had opened and the user pressed "OK."
-fullscreen Start up in full screen mode.
-terrain=name Start up in Terrain View with the indicated terrain.  This also suppresses the startup dialog.
