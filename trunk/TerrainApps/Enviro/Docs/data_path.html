The Data Path

  • If you want to put your Data directory in another location, edit your Enviro.ini file and change the first line to contain the path to the data.  The path can be relative or absolute, and should leave on a trailing slash.  For example, the default path is:
        DataPath ../Data/
    You could change it to paths such as these, depending on your platform:
        DataPath ../../Data/
        DataPath C:\MyData\
        DataPath /home/me/mydata/
  • If you have data in multiple places, you can simply put more than one "DataPath" line into the Enviro.ini file, and all the data on all the paths will be available to Enviro.
