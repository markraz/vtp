Structure Layers

Built Structure data includes buildings, fences, and other artificial structures not described by other layer types.  Structure files can be loaded and saved from VTST files.

You can also import Structure data from Shapefiles.  The Shapefile should contains arc (polygon) data, which will be interpreted as building footprints.

If you have another source of building location/footprint data that you would like to import, please let us know.

Structures Menu

  • Select Features
    • In this mode, you can select structures (by clicking or dragging the left mouse button) and edit their properties (by click the right mouse button).  Properties you can edit include:
      • Color
      • Number of Stories
      • Roof Style
  • Edit Buildings
    • In this mode, you can edit the shape and location of buildings:
      • Grab the center of a building to move it or grab a corner to resize it
      • Hold down Shift and grab the corner of a building to evenly scale it
      • Hold down Ctrl and grab the corner of a building to rotate it
