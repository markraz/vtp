Vegetation Layers

VTBuilder can import vegetation data from these formats:

  • USGS LULC (Land Use / Land Cover, usual file extension is .gir)
  • ESRI Shape files

Currently you can only display vegetation data.  Soon, you will be able to exploit the data to generate actual species distributions.

For more about how the VTP software represents vegetation, see the web page VTP Vegetation.
