Enviro

The Data Path

  • If you want to put your Data directory in another location, or in several different locations on your machine, then you should edit your Data Path.
  • The easiest way to do this is to use the Terrain Manager Dialog.

The Enviro.ini file

  • Alternately, you can edit your Enviro.ini file and change the first few lines to contain the path to the data.  A path can be relative or absolute, and should leave on a trailing slash.  For example, the default path is:
        DataPath ../Data/
    You could change it to paths such as these, depending on your platform:
        DataPath ../../Data/
        DataPath C:\MyData\
        DataPath /home/me/mydata/
  • If you have data in multiple places, you can simply put more than one "DataPath" line into the Enviro.ini file, and all the data on all the paths will be available to Enviro.
