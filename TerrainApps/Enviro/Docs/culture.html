Enviro

Working with Culture Objects

Culture is visual-simulation terminology for all things that appear on the terrain - including buildings, roads, and vegetation.  Enviro provides a number of ways of loading culture objects, interacting with them, and in some cases creating, editing and saving them.

Roads

Roads are loaded at the point a terrain is loaded.  You can tell Enviro which file contains your Road data in the Terrain Parameters dialog.

You can view, but cannot (yet) interact with Road data.

Vegetation

Vegetation (plants) are loaded at the point a terrain is loaded.  You can tell Enviro which file contains your Vegetation data in the Terrain Parameters dialog.

Creating - You can create plants using the Plants Dialog.  Menu command is "Tools: Plants", or the toolbar button that looks like a tree.  Use the dialog to select what kind of plant you want, then click on the terrain to create the plant.

Editing - To edit existing plants, use the Select tool (Menu "Tools: Select").  Click to select the plant with the 3D cursor.  Right-click and select "Properties".  This will open the Plants Dialog to edit the properties of the selected plant.

Moving - Select the plant with the Select Tool, then drag to move it on the terrain.

Deleting - Select the plant with the Select Tool, then right-click and select "Delete."

Saving - Menu command is "Terrain: Save Vegetation As..."

Structures

Structures (buildings, fences and other 3D models instances) are loaded at the point a terrain is loaded.  You can tell Enviro which file(s) contain your Structure data in the Terrain Parameters dialog.

Creating - Instance and Linear Structures (fences) can be created dynamically.  Use the "Tools: Fences" and "Tools: Instances" modes respectively to open the properties dialog, then click on the terrain to create the structure.

Editing - Buildings can be edited.  Use the Select tool (Menu "Tools: Select").  Click to select the building with the 3D cursor.  Right-click and select "Properties".  This will open the Building Dialog to edit the properties of the selected building.

Moving and Rotating - Building and Instances can be moved.  Select the structure with the Select Tool, then drag to move it on the terrain.  Instances can also be rotated: hold down the Shift key when moving the object, and it will rotate.

Deleting - Buildings and Instances can be deleted.  Select the structure with the Select Tool, then click and select "Delete."

Saving - Menu command is "Terrain: Save Built Structures As..."
